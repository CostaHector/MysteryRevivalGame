## ADDED Requirements

### Requirement: 出生房间引导
出生房间MUST引导玩家完成基础移动、获取初始线索并通过前门进入下一地点。

#### Scenario: 未调查线索直接离开
- **WHEN** 玩家在必需线索尚未调查时尝试使用前门
- **THEN** 系统阻止推进并显示当前缺失目标

### Requirement: 废弃老宅可探索空间
废弃老宅MUST包含有效导航区域、至少一个出生点、调查点、世界物品、威胁触发区和候选出口。

#### Scenario: 进入废弃老宅
- **WHEN** 地点加载完成
- **THEN** 玩家可在导航区域移动并发现至少一个可交互目标

### Requirement: 首次厉鬼遭遇编排
副本MUST在玩家获得必要预警后触发脚步鬼，并提供至少一种可取得的临时反制资源。

#### Scenario: 触发追猎
- **WHEN** 玩家进入威胁区域且满足阶段条件
- **THEN** 脚步鬼遭遇开始，目标更新为生存并寻找出口

### Requirement: 出口验证
候选出口MUST根据世界标记、关键线索和当前威胁状态判断是否为有效出口。

#### Scenario: 使用错误出口
- **WHEN** 玩家未满足真出口条件时使用候选出口
- **THEN** 系统不得通关，并提供与场景一致的异常反馈

#### Scenario: 使用真出口
- **WHEN** 玩家已完成关键目标并使用有效出口
- **THEN** 系统结束追猎并触发副本成功
